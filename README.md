# 二.必备配置

上述预览效果必要配置项：

- 前端
  - 使用如下指令安装依赖包即可 
  
  ```bash
  # 1. 安装依赖包
  pnpm install
  # 2. 启动项目
  pnpm dev
  ```
  
  - 修改后端项目后可执行如下命令快速生成接口
  
  ```bash
  pnpm openapi
  ```

- 后端
  - 根据`/sql`目录下的`/create_table.sql`创建表
  - 修改`application.yml`文件中的 `MySQL数据库连接`、`Redis数据库连接`
  - 修改`/utils/CodeUtils.java`中的邮箱配置
  - 完善阿里云oss配置
